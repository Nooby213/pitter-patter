# 피터패터 (Pitter-Patter)
운동 부족 아이들을 위한 모션 캡처 기반 활동형 게임

## 프로젝트 소개
피터패터는 아이들의 운동 부족 문제를 해결하기 위한 모션 캡처 기반 활동형 게임입니다. 

아이들은 게임을 통해 즐겁게 운동을 할 수 있으며, 부모님은 아이들의 운동량을 모니터링할 수 있습니다.
